📌 텔레그램 봇 설정 방법:
  1. 텔레그램에서 @BotFather를 검색하고 대화 시작
  2. /newbot 명령으로 봇 생성 후 토큰 발급
  3. 생성한 봇과 대화 시작 후 https://api.telegram.org/bot{TOKEN}/getUpdates에서 채팅ID 확인
  4. 환경변수 TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID 설정
