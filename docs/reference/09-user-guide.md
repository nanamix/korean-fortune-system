# 09. 사용자 가이드

> 웹 UI(`fortune-app.html`)로 사주팔자·오늘의 운세·토정비결·별자리·간지달력을 이용하고, 결과를 이메일/텔레그램으로 받는 방법을 안내합니다. 운세 용어는 초심자용으로 짧게 설명합니다.
> 관련 문서: [08. 설치 가이드](./08-installation-guide.md) · [03. 사주 계산 방법론](./03-saju-calculation-methodology.md) · [README 인덱스](./README.md)

---

## 9.1 시작하기

애플리케이션을 실행한 뒤 브라우저에서 아래 주소로 접속합니다.

```
http://localhost:18080/fortune-app.html
```

상단의 탭 버튼으로 기능을 전환합니다.

| 탭 | 기능 |
|----|------|
| 🔮 사주팔자 | 생년월일시로 정통 사주(四柱) 계산 |
| 📅 오늘운세 | 생년월일 기반 오늘의 운세 점수 |
| 📜 토정비결 | 연간 144괘 토정비결 |
| ⭐ 별자리운세 | 서양 별자리 운세 |
| 📆 간지달력 | 간지(干支) 달력 및 길일 조회 |
| 🤖 AI운세 | AI 기반 사주 해석 (AI 활성화 시) |
| 🔍 시스템 | 시스템 상태 확인 |

> 이 앱은 PWA(Progressive Web App)로, 모바일 브라우저에서 "홈 화면에 추가"하면 앱처럼 사용할 수 있습니다.

---

## 9.2 입력값의 의미

각 탭의 입력 폼에서 공통으로 사용하는 값입니다.

| 입력 항목 | 예시 | 의미 |
|-----------|------|------|
| 출생년도 | `1990` | 태어난 연도 (간지달력 API는 1900~2100 범위) |
| 출생월 | `1`~`12` | 태어난 달 |
| 출생일 | `1`~`31` | 태어난 날 |
| 출생시간 (시) | `0`~`23` | 태어난 시각의 시 (24시간제) |
| 출생분 | `0`~`59` | 태어난 시각의 분 |
| 성별 | 남성 / 여성 | 대운(大運)의 순행·역행 방향 판정에 사용 |
| 역법 | 양력 / 음력 | 입력한 생년월일이 양력인지 음력인지 |

### 양력 / 음력 (역법)

- **양력**: 우리가 일상적으로 쓰는 태양력. 주민등록상 생일이 대부분 양력입니다.
- **음력**: 달의 주기를 따르는 전통 달력. 음력 생일을 입력하면 시스템이 Time4J 기반 음양력 변환으로 양력으로 환산한 뒤 사주를 계산합니다.

정확한 사주 계산을 위해 자신의 생일이 양력인지 음력인지 반드시 올바르게 선택하세요.

### 성별과 시각

- **성별**은 대운의 진행 방향(순행/역행) 계산에 쓰이므로 결과에 영향을 줍니다.
- **출생 시각**은 사주의 마지막 기둥인 시주(時柱)를 결정합니다. 시각을 모른다면 결과의 시주 부분은 참고만 하세요.

## 9.3 사주팔자 보기

🔮 **사주팔자** 탭에서 출생년·월·일·시·분, 성별, 역법을 입력하고 계산합니다. 결과에는 4주(네 기둥), 일간, 오행 분석, 십신, 대운 등이 포함됩니다.

### 결과 해석 (초심자용 요약)

- **4주 (四柱, 네 기둥)**: 태어난 연·월·일·시를 각각 간지 두 글자(천간+지지)로 나타낸 것입니다. 연주·월주·일주·시주 4개 기둥, 총 8글자라서 "사주팔자(四柱八字)"라고 부릅니다.
- **일간 (日干, 일주의 천간 = 본원)**: 사주의 주인공에 해당하는 글자입니다. 나 자신을 상징하며, 다른 글자와의 관계를 이 글자를 기준으로 해석합니다.
- **오행 (五行)**: 목(木)·화(火)·토(土)·금(金)·수(水) 다섯 기운의 개수와 균형을 보여줍니다. 가장 강한 오행과 약한 오행이 함께 표시됩니다.
- **십신 (十神)**: 일간을 기준으로 다른 글자가 갖는 관계(예: 편재, 정관 등)를 나타냅니다. 재물·직업·인간관계 등의 성향을 읽는 지표입니다.
- **12운성 (十二運星)**: 일간이 각 지지에서 갖는 기운의 세기(예: 장생)를 단계로 표현합니다.
- **지장간 (支藏干)**: 지지 안에 숨어 있는 천간들로, 겉으로 드러나지 않는 기운을 봅니다.
- **대운 (大運)**: 약 10년 단위로 흐르는 큰 운의 흐름입니다. 시작 나이(대운수)와 순행/역행 방향이 성별·사주에 따라 정해집니다.

> 사주 계산의 상세한 알고리즘은 [03. 사주 계산 방법론](./03-saju-calculation-methodology.md) 문서를 참고하세요.

## 9.4 오늘의 운세 보기

📅 **오늘운세** 탭에서 생년월일 등을 입력하면 오늘 날짜 기준 운세를 점수로 제공합니다.

- **총점**: 오늘 운세의 종합 점수
- **분야별 점수**: 연애운 · 직업운 · 건강운 · 재물운 각각의 점수와 메시지
- **길한 방향 / 길한 색깔**: 오늘 도움이 되는 방위와 색
- **주의사항 / 조언**: 오늘 유의할 점과 조언

---

## 9.5 토정비결 보기

📜 **토정비결** 탭에서 출생년·월·일과 운세 연도를 입력합니다. 결과는 144괘(상괘8×중괘6×하괘3) 중 하나에 배정됩니다.

- **괘 번호 / 괘 이름 / 괘 기호**: 배정된 괘 정보
- **총점 · 요약 · 상세 운세**: 한 해의 전반적 흐름
- **길한 달 / 조심할 달**: 그 해 유리한 달과 주의할 달
- **조언**: 한 해를 위한 종합 조언

토정비결(土亭祕訣)은 조선시대 이지함이 지었다고 전해지는 연간 신년 운세입니다.

---

## 9.6 별자리 운세 보기

⭐ **별자리운세** 탭에서 생년월일과 대상 날짜를 입력합니다. 생일로 서양 12별자리 중 하나가 자동 판정됩니다.

- 연애·직업·건강·재물 분야별 점수와 메시지
- 행운의 숫자 · 색깔 · 보석
- 별자리별 성격 설명

---

## 9.7 간지달력 보기

📆 **간지달력** 탭에서 연도와 월을 입력하면 해당 월의 간지 달력을 실제 달력 형태로 볼 수 있습니다. 각 날짜의 간지와 길일 정보를 확인할 수 있습니다. (조회 가능 범위: 1900~2100년)

브라우저에서 직접 접근할 수도 있습니다.

```
http://localhost:18080/api/calendar/view/current      # 이번 달
http://localhost:18080/api/calendar/view/2025/6        # 2025년 6월
```

## 9.8 이메일 / 텔레그램으로 결과 받기

사주팔자·오늘운세·토정비결·별자리 각 탭에는 결과를 발송하는 옵션이 있습니다. 입력 폼 하단에서 다음을 지정합니다.

| 항목 | 설명 |
|------|------|
| 수신자 이름 | 메시지에 표시될 이름 (예: 홍길동) |
| 이메일 주소 | 이메일 발송 대상 |
| 텔레그램 채팅ID | 텔레그램 발송 대상 (숫자 ID) |
| 발송 방법 | 이메일 / 텔레그램 / 둘 다 |

- **이메일 발송**: 서버에 SMTP(메일 계정)가 설정되어 있어야 합니다.
- **텔레그램 발송**: 서버에 텔레그램 봇 토큰이 설정되어 있어야 하며, 수신할 채팅ID가 필요합니다.

> 발송 기능은 서버 측 자격증명 설정에 의존합니다. 설정 방법은 [08. 설치 가이드 §8.4.3](./08-installation-guide.md) 을 참고하세요. 자격증명이 없으면 발송은 실패하지만 운세 계산 자체는 정상 동작합니다.

### 텔레그램 채팅ID 확인 방법

1. 텔레그램에서 봇과 대화를 시작합니다.
2. 봇 관리자에게 채팅ID를 문의하거나, `@userinfobot` 같은 봇으로 자신의 숫자 ID를 확인합니다.

## 9.9 AI 운세 (선택)

🤖 **AI운세** 탭은 서버에서 AI 기능이 활성화되어 있을 때만 실제 AI 해석을 제공합니다. 비활성 상태에서는 "AI 서비스가 활성화되지 않았습니다" 응답이 나올 수 있으며, 이 경우에도 기본 사주 계산 기능은 정상적으로 사용할 수 있습니다. AI 활성화 방법은 [08. 설치 가이드 §8.4.4](./08-installation-guide.md) 를 참고하세요.
